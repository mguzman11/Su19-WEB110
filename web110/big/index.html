Welcome

Welcome to my final project for WEB 110 Summer Quarter 2019! This page is an example of all that I have learned throughout this quarter. Creating this website was definitely my favorite assignment. I was able to use my research, critical thinking, and writing skills to explore topics in web design, learn different responsive tools, such as lightbox and google tools, and I was able to customize my CSS styles to my personal taste.

As much I do enjoy web design, my goal isn't to become a full fledge web designer, but take this knowledge and use it for side projects. Knowing the ins and outs of great web design will allow me to be a better software developer and let me embrace my creative side from time to time. Please take your time and see all that I have learned this quarter!
